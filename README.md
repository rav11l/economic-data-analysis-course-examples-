# Примеры кода из курса «Анализ данных для экономистов»

Этот репозиторий содержит рабочие примеры кода, приведённые в курсе **«Анализ данных для экономистов: практический курс (обновлённая программа 2026)»**. Код оформлен в виде отдельных скриптов, готовых к запуску.

Курс на Stepik: https://stepik.org/course/274415

## О курсе

Курс построен вокруг идеи **гибридного анализа** — сочетания прогностической мощи машинного обучения с интерпретируемостью структурных эконометрических моделей. Задача — научить экономиста не просто применять инструменты, а проектировать модульные системы прогнозирования, работать с реальными российскими данными (Росстат, Мосбиржа), критически оценивать результаты с помощью современных метрик и статистических тестов и формулировать обоснованные рекомендации для лиц, принимающих решения.

В основе — **архитектурный подход**: система разделяется на независимые модули (сбор данных, подготовка, моделирование, оценка, визуализация), взаимодействующие через чётко определённые интерфейсы (API). Это обеспечивает гибкость, масштабируемость и лёгкость поддержки.

- **Аудитория:** экономисты, финансовые аналитики, специалисты по данным, студенты экономических направлений.
- **Уровень:** базовый + продвинутый, с акцентом на практику.
- **Формат:** 70% практики, 30% теории; 7 модулей со сквозным проектом.
- **Итог:** аналитический отчёт или дашборд с модульной прогнозной системой для ключевых макропоказателей России.

## Чему вы научитесь

- Находить, очищать и подготавливать экономические данные из открытых источников.
- Применять статистические и эконометрические методы для анализа данных.
- Строить прогнозные модели с помощью ARIMA и машинного обучения.
- Визуализировать данные и создавать интерактивные дашборды.
- Готовить аналитические отчёты для принятия экономических решений.
- Работать с инструментами: Excel, Python (Pandas, NumPy), AW BI.

## Содержание репозитория

## Где искать экономические данные

Справочник источников, разбираемых в Модуле 1:

**Российские официальные источники**
- Росстат (rosstat.gov.ru) — ВВП, инфляция, безработица, демография, региональные показатели.
- Центральный банк РФ (cbr.ru) — ключевая ставка, курсы валют, инфляция (ИПЦ), кредиты и депозиты.
- Министерство финансов (minfin.gov.ru) — бюджет, государственный долг.
- ЕМИСС (fedstat.ru) — единая межведомственная система с поиском и API.

**Международные источники**
- Всемирный банк (worldbank.org) — данные по всем странам, бесплатный API.
- МВФ (imf.org) — макроэкономические показатели, International Financial Statistics.
- ОЭСР (oecd.org) — статистика развитых стран.

**Финансовые данные**
- Московская биржа (moex.com) — котировки, индексы, облигации (ОФЗ), курсы валют.
- MOEX ISS — открытый REST API для исторических и текущих данных.
- `aiomoex` — асинхронный Python-клиент для MOEX ISS на базе `asyncio` и `aiohttp`, ускоряющий загрузку в десятки раз.

**Альтернативные данные**
- Сайты вакансий (hh.ru, SuperJob) — спрос на труд, зарплаты, навыки.
- Новостные ленты и соцмедиа — тональность сообщений опережает официальные индексы настроений.
- Маркетплейсы (Wildberries, Ozon) — цены, объёмы продаж, потребительские предпочтения.
- Данные GPS-трекеров и транзакций — мобильность и потребительская активность.
