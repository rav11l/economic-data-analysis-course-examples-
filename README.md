# Примеры кода из курса «Анализ данных для экономистов»

Этот репозиторий содержит рабочие примеры кода, приведённые в курсе «Анализ данных для экономистов: практический курс (обновленная программа 2026)». Код оформлен в виде отдельных скриптов, готовых к запуску.

## Содержание

- **module1/fetch_imoex.py** – получение исторических данных индекса IMOEX с MOEX через библиотеку `aiomoex`.
- **module4/volatility_var.py** – расчёт волатильности и Value-at-Risk (VaR) для портфеля акций.
